Biography


Hi folks. I'm Paul. I'm a front-end developer who loves the web. I'm always thinking a lot about how to make you more productive with tools that improve your workflow helpyou make slicker, sexier, mobile web sites and webapps.


I work on the Google chrome team focused on Developer Tooling. Chrome DevTools and everything else Chrome can do to make you a productive web designer.


I've developed tools like Modernizr, Yeoman, CSS3 Please and HTML5 Boilerplate, curated HTML5 Rocks, and co-created projects like HTML5 Please, motherffingtextshadow, motherffinghsl, Move the Web Forward, Mother-iffing Animated GIF, w3Fools, chromestatus.com, HTML5 Readiness, Front-end Coding Standards, Type Rendering Project.
